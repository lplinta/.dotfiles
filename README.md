## Instalação

1.  **Se necessário, torne o script executável:**
    ```bash
    chmod +x install.sh
    ```

2.  **Execute o script de instalação:**
    ```bash
    ./install.sh
    ```
3. **Configurar o tmux**
   
   Com o tmux rodando:
   ```bash
    tmux source ~/.tmux.conf
    ```
   Para instalar os plugins: `prefix` + <kbd>I</kbd>
